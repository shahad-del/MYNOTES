    Here are some basic argument specifiers you should know:
  1. %s - String (or any object with a string representation, like numbers)
  2. %d - Integers
  3. %.f - Floating point numbers with a fixed amount of digits to the right of the dot.
  4. %x/%X - Integers in hex representation (lowercase/uppercase)
