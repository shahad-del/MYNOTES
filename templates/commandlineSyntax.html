Let’s say that you’re in the command line and that you need to move into a folder that’s far away, something like ~/Documents/OdinProject/foundations/javascript/calculator/.  we’re way too lazy for that! Basically, by hitting Tab, the command line will automatically complete commands that you’ve started typing once there’s only one option.

we can create a new folder using the mkdir command, which stands for "Make Directory”.Type mkdir testdir and press ENTER

To create multiple nested directories at once, use the -p flag to create a whole chain of directories like this:  

mkdir -p directory1/directory2/directory3

To browse through different directories in the file system. We can do this via the cd command, which stands for "Change Directory”.

Type cd .. and press ENTER. The .. tells the Command Line to browse backwards to the parent directory. 

    
        
  1. cd .. (two dots). The .. means “the parent directory” of your current directory, so you can use cd .. to go back (or up) one directory.
    2. 
        
  2. cd ~ (the tilde). The ~ means the home directory, so this command will always change back to your home directory (the default directory in which the Terminal opens).

To create a new empty file in the current directory Type: 

touch newfile1.txt 

and press ENTER. Now we’ll copy that file from one folder to another using the cp command.Type:

cp newfile1.txt testdir
 

and press ENTER.

We can also move files instead of copying using the mv command. 

Type touch newfile2.txt and press ENTER to create a new file.
Next, type mv newfile2.txt testdir and press ENTER to move the file into the "testdir" folder.

The mv command can also be used to rename files.To do that, type 

touch newfile3.txt and press ENTER to create a new file. Then type 
 mv newfile3.txt cheese.txt
ENTER to update the file’s name.




Finally, we can delete files and folders using the rm command.Type:
rm cheese.txt
 and press ENTER to remove the file.Type: 
rm -rf testdir
 and press ENTER to remove the "testdir" directory and its contents.
