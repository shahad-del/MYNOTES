If you rely on the order in which they've been recorded you should either use collections.OrderedDict if using Jinja2 in Python script, or you can apply dictsort filter in your template to order your dictionary by key or value. To sort by key: 


       {% for k, v in my_dict | dictsort %}
To sort by value: 

    {% for k, v in my_dict | dictsort(by='value') %}
